**«Сотня кроликов»** — это коллектив художников, документирующий низкотехнологичные решения и стремящийся к более устойчивому будущему. Они живут и работают на борту 10-метровой парусной лодки *Pino* в отдалённых районах мира, чтобы на собственном опыте изучать, как технологии деградируют за пределами развитых регионов.

Проект посвящён:

* минималистичному программированию
* независимому творчеству
* устойчивому дизайну
* документированию путешествий и быта

Архивирование **100r.co** необходимо для обеспечения долговременной сохранности контента, который регулярно обновляется и потенциально может исчезнуть из публичного доступа.

# Wpull

Этот раздел содержит результаты архивации сайта при помощи инструмента массовой загрузки **Wpull**.
Для архивации использовались:

* рекурсивный обход страниц
* сохранение служебных данных
* логирование запросов
* формирование WARC, CDX и служебной базы данных

## Результаты

* **Общий размер архива:** 175 МБ
* **Основной WARC:** `100r-co-00000.warc.gz`
* **Meta WARC:** `100r-co-meta.warc.gz`
* **Индекс CDX:** `100r-co.cdx`
* **База данных:** `site-archive.db`

* **Всего URL обработано:** 3744
* **Успешно скачано:** 2091
