# Архив сайта MarkdownGuide.org

<img width="1080" height="225" alt="image" src="https://github.com/user-attachments/assets/e5dfc616-8405-4879-a886-023e198ce903" />

**MarkdownGuide.org** — один из наиболее авторитетных и широко используемых онлайн-ресурсов, посвящённых обучению синтаксису Markdown. Сайт включает подробные руководства, примеры базового и расширенного синтаксиса, удобную шпаргалку, документацию API и подборку инструментов, поддерживающих Markdown.

Архивирование этого ресурса важно потому, что:

* сайт служит ключевым справочником для разработчиков, исследователей и студентов;
* структура и содержание обновляются со временем, что делает актуальные версии нестабильными;
* многие страницы загружают внешние библиотеки и инструменты, которые могут исчезнуть или измениться;
* руководство по Markdown представляет ценность как образовательный и технический артефакт современной цифровой культуры.

Сохранение полного набора статей и документов позволяет обеспечить независимый доступ к знаниям вне зависимости от состояния сайта в будущем.

# ArchiveReady

Этот раздел содержит результаты анализа архивируемости сайта **MarkdownGuide.org** с помощью сервиса **ArchiveReady**, который оценивает ресурсы по методологии **CLEAR** (Cohesion, Accessibility, Metadata, Standards Compliance).

## Результаты

* **Общий рейтинг архивируемости:** **79%**
* **Accessibility:** 62%
* **Cohesion:** 90%
* **Metadata:** 100%
* **Standards Compliance:** 65%

Сайт корректно формирует все важные HTTP-заголовки (Metadata 100%). Это обеспечивает устойчивое воспроизведение сохранённых страниц, делает архив предсказуемым и технически корректным.

MarkdownGuide — хороший пример статического, структурированного сайта (Cohesion 90%): отсутствует тяжёлый динамический контент, большая часть ресурсов размещена локально.

Средняя доступность (Accessibility 62%) объясняется несколькими факторами: не все изображения имеют альтернативный текст, некоторые элементы интерфейса ориентированы на визуальное восприятие.

Хотя HTML и CSS в целом валидны (Standards Compliance 65%), сервис обнаружил частичное несоответствие стандартам в отдельных стилевых и структурных элементах.



Сайт **MarkdownGuide.org** демонстрирует высокую архивируемость благодаря своей статической природе, качественным метаданным и хорошей связности страниц.
Имеющиеся ограничения (легкие проблемы со стандартами и доступностью) не препятствуют воспроизведению архива и не снижают ценность сохранённого материала.

В целом оценка **79%** говорит о том, что ресурс **хорошо подготовлен к долгосрочному хранению** и будет корректно восстанавливаться архиваторами.
